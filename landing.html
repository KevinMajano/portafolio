Ir a Web

Landing page para una campaña vocacional de los salesianos realizada anualmente, contiene recursos descargables como artes, pdfs, o plantillas de uso

Una landingpage conectada con un link de google drive, para la descarga de todos los archivos necesarios y tambien el agregar dinamicamente links de youtube o tiktok

  • Nuxt.js Uso para que el front-end sea mas optimo
  • Strapi Generar un backend de manera inmediata y que la escalabilidad sea en menor tiempo posible
  • Docker Uso de docker para montar el sitio
  • Figma Se basa en una maquetación previamente diseñada en Figma
  • Sass Uso de sass para mejorar la velocidad de codificacion
