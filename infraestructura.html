Detalles del proyecto

De los proyectos principales fue el establecer una infraestructura que soporte multiples sitios web, agregando aplicaciones y un sistema de facturacion para parroquia que estaria contenido en la instancia.

Para poder establecer un buen marco de los requerimientos vs los precios, se utilizaron herramientas como la calculadora de AWS para estimacion de precios, agregando los servicios pertinentes a ese requerimiento.

  • Uso de Instancia EC2 Montaje de una MV con sistema de la distro Linux.
  • Balanceador de carga Uso y configuracion de un balanceador de carga.
  • Backup de instancia Configuracion del servicio de backup.
  • Certificado SSL Uso de los servicios para generar un certificado de seguridad.
  • WAFConfiguracion de un firewall para aumentar la seguridad y no comprometer datos sensibles.
